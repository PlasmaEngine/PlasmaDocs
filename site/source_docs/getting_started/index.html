Getting Started

Before diving into the Plasma Source, it is important to become familiar with Plasma itself as a game engine. Please visit getting_started.

Requirements

  • Windows 7 or higher
  • Visual Studio 2019
  • Node.JS

Building

To build the engine you will need to run the following:

git submodule update --init --recursive
npm install
node index.js cmake --builder="Visual Studio 16 2019"
