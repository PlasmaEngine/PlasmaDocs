# UseAmplitudeModulation : boolean

If this property is true, the node will apply amplitude modulation (multiply the audio input with a unipolar sine wave with values from 0.0 to 1.0). If false, the node will apply ring modulation (multiply the input with a bipolar sine wave with values from -1.0 to 1.0). ``` lang=cpp, name=Lightning var UseAmplitudeModulation : Boolean
