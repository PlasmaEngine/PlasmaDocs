# Paused : boolean

Setting this Property to true will pause a currently playing SoundCue. Setting it to false will resume playback. ``` lang=cpp, name=Lightning var Paused : Boolean

# Positional : boolean

If this property is true the SoundCue will be played positionally (heard at a specific location by SoundListeners) through the SoundEmitter component on the same object. If false, the SoundCue will be played through the SoundSpace, and will NOT be affected by any SoundEmitter settings. ``` lang=cpp, name=Lightning var Positional : Boolean

# StartPlaying : boolean

If this property is true the SoundCue will begin playing as soon as the object is created. ``` lang=cpp, name=Lightning var StartPlaying : Boolean
