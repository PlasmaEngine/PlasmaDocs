# NoteOn : Void

Starts playing a new note. The first parameter is the MIDI note value (range is 0 to 127), and the second is the volume modification that should be applied to this note (a value of 1.0 does not change the volume, while 0.0 would be silence). |Name|Type|Description| |---|---|---| |midiNote|real| | |volume|real| | lang=cpp, name=Lightning function NoteOn(midiNote : Real, volume : Real)
