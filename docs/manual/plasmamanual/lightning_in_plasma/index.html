This guide is meant for those who are already familiar with another programming language and Object Orientation Programming principles. Its main purpose is to help orient new users to its syntactic structure. As this language is made to be embedded in C++, the differences between Lightning and C++ will be highlighted, although users of ActionScript 3.0 may find it has a familiar syntax.
